Essential Technical Terms

To understand how mobile browsing works, we must define these 5 core concepts:

Term Definition Security Impact
Rendering Engine The software (like WebKit) that interprets HTML/CSS into pixels. Vulnerabilities here can lead to remote code execution.
Sandboxing Restricting the browser's access to the device's OS and files. Prevents a malicious site from stealing your photos or contacts.
User Agent A string sent to servers to identify the browser and OS. Can be spoofed to bypass mobile-only restrictions.
PWA Web apps that function like native apps (offline mode, icons). Requires careful permission handling for hardware access.
HTTPS Encrypted data transfer between the browser and the server. Critical for mobile users on public Wi-Fi (like at a cafe).
